Protéger des sites contre le Hotlinking (Windows)

Regardez le tutoriel vidéo

Le Hotlinking (également appelé file leeching, remote linking, direct linking, vol de bande passante ou banditisme de bande passante) est un terme utilisé pour décrire une opération frauduleuse. Elle consiste à créer un lien direct entre l'une des pages Web d'un propriétaire de domaine et des images (ou autres fichiers multimédia) sur les pages Web d'un autre propriétaire de domaine, généralement en utilisant un marqueur <IMG>. Si vos domaines sont victimes de Hotlinking, vous rencontrerez probablement un problème de consommation excessive de bande passante.

Pour protéger un site Web contre le vol de bande passante :

  1. Allez dans Sites Web & Domaines et repérez le nom de domaine de votre site.
  2. Cliquez sur Protection contre le Hotlink.
  3. Cliquez sur Activer pour activer la protection contre le Hotlinking.
  4. Dans le champ Extensions des fichiers protégés indiquez les extensions des fichiers que vous souhaitez protéger contre le Hotlinking, par exemple .jpg, .bmp, etc. Si vous saisissez une liste d'extensions, séparez-les par des espaces.
  5. Si vous voulez autoriser la liaison directe pour certains sites à vos fichiers, saisissez les adresses des sites dans le champ Adresses de sites Web de confiance et cliquez sur Ajouter.
  6. Cliquez sur OK.
