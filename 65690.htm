Protéger les boîtes mail contre les spams

Voir le tutoriel vidéo

Pour activer le filtre anti-spam pour une boîte mail :

  1. Allez dans Mail > adresse mail > onglet Antispam.
  2. Cochez Activer le filtre anti-spam pour cette adresse mail.
  3. Indiquez la procédure à suivre pour les messages classés en tant que spam.
    • Si vous voulez filtrer les mails avec le logiciel installé sur votre ordinateur local, activez l'option Marquer comme spam en ajoutant le texte suivant dans l'objet du message puis précisez comment le filtre anti-spam doit marquer les mails reconnus comme étant des spams. "X-Spam-Flag: YES" et "X-Spam-Status: Yes" sont ajoutés par défaut à la source du message, et si vous le souhaitez, le filtre anti-spam ajoutera également un texte spécifique au début de la ligne Objet et dans le corps du message.
    • Si vous êtes sûr que votre filtre anti-spam est précis, vous pouvez le configurer pour qu'il supprime automatiquement tous les messages entrants reconnus comme spams. Pour ce faire, sélectionnez l'option Supprimer tous les spams.
    • (Linux uniquement) Si vous accédez à votre boîte mail via le protocole IMAP et si vous voulez que le filtre anti-spam déplace tous les messages considérés comme du spam dans le dossier IMAP appelé Spam, sélectionnez l'option Déplacer le spam vers le dossier Spam.

      Comme tous les messages sont automatiquement supprimés du dossier Spam après 30 jours, vous devrez régulièrement vérifier le contenu du dossier Spam pour vous assurer que vous n'avez pas manqué de messages importants et pour replacer tous les messages non spam dans le dossier Boîte de réception.

      Lorsque l'option Déplacer le spam dans le dossier Spam est cochée, vous pouvez former le filtre anti-spam et améliorer sa précision en déplaçant les spams de la Boîte de réception vers le dossier Spam et les messages non-spams du dossier Spam vers la Boîte de réception.

  4. Si le filtre anti-spam manque certains spams ou renvoie de nombreux faux positifs, vous devez ajuster la sensibilité du filtre. Pour cela, cliquez sur Afficher les paramètres avancés et saisissez la valeur souhaitée dans le champ Intensité du filtre anti-spam.

    Par défaut, la valeur 7 convient à la plupart des situations. Cependant, si trop de spam n'est pas filtré, diminuez cette valeur d'un et vérifiez si le volume de spam non filtré diminue. Si non, diminuez encore la valeur d'un. Dans le cas contraire, si des mails non spams sont classés comme tel, vous devez augmenter la valeur de la sensibilité. Dans tous les cas, il convient d'augmenter ou de diminuer la valeur d'un cran à la fois. En effet, de trop grands écarts risquent d'entraîner des modifications radicales dans le comportement du filtre anti-spam.

    Remarque : pour continuer à améliorer la précision du filtre anti-spam, vous pouvez vouloir former votre filtre anti-spam avec les mails que vous recevez, comme décrit plus loin dans cette section.

  5. Si vous voulez être sûr que vous ne raterez pas de mails d'expéditeurs spécifiques, saisissez les adresses mail ou les noms de domaine dans le champ Liste blanche.

    Mettez une adresse par ligne ou séparez les adresses par une virgule, un point-virgule ou un espace. Vous pouvez utiliser un astérisque (*) pour remplacer un certain nombre de lettres et un point d'interrogation (?) pour remplacer une seule lettre. Par exemple : address@mycompany.com, user?@mycompany.com, *@mycompany.com. Si vous définissez *@mycompany.com, toutes les adresses mail du domaine de messagerie mycompany.com seront ajoutées à la liste blanche.

  6. Si vous ne voulez pas recevoir de mails d'expéditeurs ou de domaines spécifiques, saisissez les adresses mail ou les noms de domaine dans le champ Liste noire.

    Mettez une adresse par ligne ou séparez les adresses par une virgule, un point-virgule ou un espace. Vous pouvez utiliser un astérisque (*) pour remplacer un certain nombre de lettres et un point d'interrogation (?) pour remplacer une seule lettre. Par exemple : address@spammers.net, user?@spammers.net, *@spammers.net. Si vous indiquez *@spammers.net, tout le domaine de messagerie spammers.net sera bloqué.

  7. Si vous utilisez un compte client basé sur Windows, vous pouvez également indiquer des langues et des jeux de caractères de confiance. Les mails écrits dans les langues indiquées et avec les jeux de caractères définis passeront le filtre anti-spam et ne seront pas marqués en tant que spam.
  8. Sur les comptes client basés sur Windows, vous pouvez indiquer les adresses IP des ordinateurs ou réseaux à partir desquels le mail doit toujours être accepté. En indiquant des adresses réseau, saisissez une adresse et ajoutez un masque de réseau après un slash. Par exemple, 192.168.10.10/24.
  9. Cliquez sur OK.

Vous pouvez améliorer la précision de la détection des spams si le filtre anti-spam SpamAssassin est activé sur le serveur pour votre compte et si vous accédez à votre boîte mail via le protocole IMAP.

Pour augmenter la précision de détection des spams :

  1. Accédez à votre boîte mail avec la messagerie Web ou un programme client de messagerie sur votre ordinateur.
  2. Passez en revue les messages dans votre dossier Boîte de réception. Déplacez tous les spams dans le dossier Spam. Le filtre anti-spam SpamAssassin reconnaîtra alors les spams plus efficacement.
