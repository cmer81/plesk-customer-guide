Accéder à votre boîte mail depuis un iPhone

Les instructions indiquées dans cette section ont été vérifiées sur un iPhone 5s. 

Pour configurer un compte mail sur un iPhone :

  1. Touchez Réglages. Faites défiler la page vers le bas et touchez Mail, Contacts, Calendriers et sélectionnez Ajouter un compte.


  2. Dans la liste affichée, sélectionnez Autre. Puis, touchez Ajouter un compte Mail.



  3. Sur l'écran Nouveau, saisissez les informations suivantes :
    • Nom. Indiquez le nom qui sera affiché dans vos mails.
    • Adresse. Dans ce champ, indiquez votre adresse mail complète. Exemple : admin1@exemple.com.
    • Mot de passe. Saisissez le mot de passe de votre compte mail.
    • Description. Saisissez la description du compte (facultatif).

  4. Touchez Suivant. En haut de l'écran suivant, sélectionnez IMAP (si vous voulez conserver les copies des mails reçus sur le serveur) ou POP. Ensuite, pour les serveurs entrants et sortants, complétez ces infos :
    • Nom d'hôte. Saisissez votre nom de domaine, par exemple :  exemple.com.
    • Nom d'utilisateur. Saisissez le nom de votre compte mail, par exemple : admin@exemple.com.
    • Mot de passe. Saisissez le mot de passe de votre compte mail (facultatif pour le serveur de messagerie sortant ou serveur d'envoi).


  5. Une fois que vous avez ajouté la configuration, touchez Enregistrer pour continuer. La connexion au serveur indiqué sera vérifiée. Si la connexion fonctionne, vous verrez l'écran depuis lequel vous pouvez choisir les informations que vous voulez synchroniser avec votre téléphone.



  6. Une fois que vous avez indiqué toutes les informations, cliquez sur Enregistrer pour les sauvegarder. Votre iPhone vérifiera les informations de votre compte. Une fois que vous avez terminé, vous serez redirigé vers l'écran Mail. Votre compte est désormais listé dans la section Comptes.

Remarque : si vous rencontrez des problèmes de synchronisation avec vos messages, vous pouvez configurer manuellement la correspondance des dossiers. Pour cela, sélectionnez le compte que vous venez de créer et allez dans la section Avancé. Dans cette section, vous pouvez choisir une boîte mail (Brouillons, Envoyés, Corbeille, Archives) et sélectionner le dossier correspondant sur le serveur. De cette manière, vos dossiers seront synchronisés où que vous soyez et quelle que soit la manière dont vous accédez à vos mails.
