Utiliser des serveurs de messagerie externes (comme Gmail)

Si vous devez configurer un serveur de messagerie externe pour traiter les mails de votre domaine, procédez comme suit :

  1. Allez dans Mail, puis cliquez sur Paramètres de la messagerie.
  2. Cochez la case en regard du domaine pour lequel vous devez configurer le serveur de messagerie externe et cliquez sur Activer/Désactiver les services.
  3. Dans le menu, sélectionnez Désactiver et cliquez sur OK.
  4. Allez sous Sites Web & Domaines. Cherchez le nom de domaine dont vous voulez gérer les paramètres DNS et cliquez sur Paramètres DNS.
  5. Cliquez sur le lien hypertexte dans la colonne Hôte correspondant à l'enregistrement de ressource MX que vous voulez modifier.
  6. Saisissez le nom de domaine de votre serveur de messagerie externe dans le champ Serveur d'échange de mail. Si vous utilisez un serveur de messagerie distant dont le nom est "mail.monnomdhôte.com", saisissez simplement "mail.monnomdhôte.com".
  7. Indiquez la priorité du serveur de messagerie externe, 0 étant la priorité la plus haute et 50, la plus faible.
  8. Cliquez sur OK, puis sur Mettre à jour.
