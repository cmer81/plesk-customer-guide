Créer un site Web

Pour minimiser vos efforts lors de la création d'un site Web, Presence Builder vous offre un jeu de thèmes prédéfinis de sites Web. Lorsque vous commencez à créer un site Web, choisissez le thème qui vous convient le mieux et Presence Builder génère un site Web avec le contenu approprié pour ce thème. Par exemple, il inclut un template de liste de prix dans une boutique ou une galerie d'images dans un site d'artiste. Si vous n'arrivez pas à trouver un thème adapté, vous pouvez contacter votre fournisseur de services pour lui demander de créer le template correspondant et de l'ajouter à la liste des thèmes disponibles.

Au début de la création d'un site Web, Presence Builder vous invite également à indiquer les informations sur vous-même ou votre société. Presence Builder les ajoutera automatiquement à certaines pages où elles pourraient être appropriées, par exemple, dans les pages À propos et Nous contacter.

Remarque : Presence Builder ne stockera ou n'utilisera aucune des informations que vous indiquerez, sauf dans les pages de votre site Web. Vous pouvez modifier ou supprimer plus tard ces informations.

Pour commencer à créer un site Web dans Presence Builder :

  1. Allez dans Sites Web & Domaines et cliquez sur le nom de domaine souhaité.
  2. Sous Presence Builder, cliquez sur Créer un site.
  3. Sélectionnez un thème de site Web.
  4. Tapez le nom du site Web et sélectionnez sa langue.
  5. Indiquez les informations pour pré-remplir votre site Web.
  6. Cliquez sur Créer le site.

Une fois que vous avez quitté l'éditeur, si vous voulez modifier le site Web que vous avez créé avec Presence Builder, allez dans Sites Web & Domaines. Cliquez sur le nom de domaine et sur Éditer le site Web.
