Plesk en tant que Serveur DNS maître

Lorsque vous créez un domaine dans Plesk, une zone DNS avec tous les enregistrements DNS requis est créée automatiquement. Toutefois, vous pouvez ajouter, modifier ou supprimer des enregistrements DNS si l'hébergeur vous a accordé le droit correspondant.

Gestion des enregistrements DNS

Pour voir les enregistrements actuels d'une zone DNS, allez sous : Sites Web & Domaines > Paramètres DNS. Depuis cet écran, vous pouvez également ajouter, modifier et supprimer des enregistrements.

Pour ajouter un enregistrement, cliquez sur Ajouter un enregistrement. Pour modifier un enregistrement, cliquez sur son nom. Pour supprimer un enregistrement, cochez la case à côté de son nom et cliquez sur Supprimer. Remarque : supprimer certains enregistrements risque d'affecter le fonctionnement de votre domaine. Par exemple, si vous supprimez l'enregistrement A, la résolution ne fonctionnera plus. Si vous avez modifié des éléments de la zone DNS qui affectent le fonctionnement de votre site Web, vous pouvez restaurer les enregistrements DNS par défaut. Pour cela, cliquez sur Rétablir le paramétrage initial. Cela permet de réinitialiser les modifications de la zone DNS et de la recréer conformément aux paramètres définis dans le template DNS valable pour l'ensemble du serveur. Notez que tous les enregistrements personnalisés que vous avez ajoutés à la zone seront perdus.

La zone DNS inclut également l'enregistrement SOA. Pour afficher et modifier les valeurs de l'enregistrement SOA, cliquez sur Sites Web & Domaines > Paramètres DNS et allez dans l'onglet SOA. Si vous cochez la case Utiliser le format du numéro de série recommandé par IETF et RIPE, Plesk change le format d'enregistrement des numéros de série SOA : autrement dit, l'horodatage UNIX est converti au format YYYYMMDDNN recommandé par RIPE. Le format YYYYMMDDNN est requis par de nombreux registraires de domaines, notamment européens. Si votre registraire refuse le numéro de série SOA, essayez d'activer cette option.

Remarque : si les numéros de série SOA sont stockés au format recommandé par IETF et RIPE, notez que le maximum de changements d'un enregistrement SOA est de 98. Si vous avez modifié votre enregistrement SOA dans un domaine 98 fois au cours de la même journée, alors vous ne pouvez plus changer le numéro de série SOA. Si vous cliquez sur le bouton Par défaut, tous les enregistrements de la zone DNS seront supprimés et le service DNS de la zone DNS ne pourra pas être activé/désactivé ou basculer du mode esclave au mode maître.

Application des modifications du template DNS (Plesk pour Linux)

Plesk pour Linux vous permet de synchroniser les enregistrements du template DNS valable pour tout le serveur avec la zone DNS de votre domaine. La synchronisation est réalisée avec les règles suivantes :

  • Les enregistrements modifiés par l'utilisateur restent toujours intacts (ne sont jamais modifiés ou supprimés).
  • Les enregistrements ajoutés dans le template sont ajoutés à la zone.
  • Les enregistrements supprimés du template sont supprimés de la zone (les enregistrements que vous avez modifiés ne sont pas supprimés).
  • Les enregistrements modifiés dans le template sont modifiés de la zone (les enregistrements que vous avez modifiés ne sont pas supprimés).

Pour synchroniser les enregistrements du template DNS et ceux votre zone DNS, cliquez sur Sites Web & Domaines > Paramètres DNS, puis cliquez sur le bouton Appliquer le template DNS.
