Sites Web multilingues

Selon votre pack de services et la politique de votre hébergeur, vous pouvez proposer un site Web multilingue. Pour ce faire :

  1. Allez sous l'onglet Paramètres > Langues. Cliquez sur Ajouter une langue et sélectionnez la langue que vous souhaitez ajouter. Cliquez sur OK.

    Une copie de votre site Web est créée dans une autre langue. Il est synchronisé avec la version principale :

    • Quand vous ajoutez une page à votre site ou insérez un module dans une page, cette page ou ce module sont insérés dans le site principal et dans la copie du site dans les autres langues.
    • Quand vous supprimez une page de votre site ou un module d'une page, cette page ou ce module sont supprimés du site principal et des copies du site dans les autres langues.
  2. Allez dans l'onglet Modules, sélectionnez le module Switcheur de langues. Glissez-le à l'emplacement souhaité sur la page, par exemple, dans la zone DESIGN. Choisissez le style du menu de langues. Cliquez sur OK.
  3. Pour modifier un texte, passez dans la langue souhaitée, soit en cliquant sur les liens dans le module Switcheur de langues, soit en cliquant sur les drapeaux affichés dans les barres d'outils des modules.
