Ressources allouées et utilisées

Si vous êtes abonné à plusieurs packs de services en même temps, vous avez plusieurs abonnements. Pour chaque abonnement, les types suivants de ressources allouées apparaissent dans Plesk sous Compte > Ressources :

  • Espace disque. Il s'agit de l'espace disque total alloué à votre compte avec l'abonnement. Ce volume inclut toutes les données associées à vos sites Web, comptes mail, applications, sauvegardes et fichiers de log.
  • Trafic. Il s'agit de la somme totale de données en mégaoctets qui peuvent être transférées tous les mois depuis tous vos sites Web.
  • Domaines. Il s'agit du nombre de sites Web pouvant avoir des noms de domaine de deuxième niveau distincts, comme, par exemple, exemple.com.

    Pour en savoir plus sur la configuration des sites Web, consultez les sections Configurer votre premier site Web et Ajouter des domaines.

  • Sous-domaines. Il s'agit du nombre d'autres sites Web pouvant avoir des noms de domaine de troisième niveau distincts, comme, par exemple, news.exemple.com.

    Pour en savoir plus sur la configuration de sous-domaines, consultez la section Ajouter des sous-domaines.

  • Alias de domaine. Il s'agit du nombre de noms de domaine supplémentaires pouvant être configurés pour pointer vers l'un de vos sites. Par exemple, exemple.fr et exemple.de peuvent tous les deux pointer vers exemple.com.

    Pour en savoir plus sur la configuration des alias de domaine, consultez la section Ajouter des alias de domaines.

  • Boîtes mail. Il s'agit du nombre de boîtes mail pouvant être créées dans tous vos sites Web.

    Pour en savoir plus sur la création de boîtes mail, reportez-vous au chapitre Messagerie.

  • Volume de la boîte mail. Il s'agit du total d'espace disque que peut occuper la boîte mail.
  • Quota total des boîtes mail. Il s'agit du total d'espace disque utilisable par toutes les boîtes mail dans tous vos domaines. Cette option n'est disponible que sur les comptes client Windows.
  • Listes de diffusion. Il s'agit du nombre total de listes de diffusion configurables dans tous vos sites Web. Remarque : il existe des listes de messagerie et des groupes de messagerie ayant le même objectif mais qui sont légèrement différents en termes de fonctionnalités. Les listes de diffusion prennent en charge l'archivage et la pré-modération des messages alors que les groupes de messagerie peuvent uniquement servir à envoyer un message à plusieurs destinataires en même temps.

    Pour en savoir plus sur la configuration et l'utilisation des listes de diffusion, reportez-vous à la section Utiliser les listes de diffusion.

  • Comptes FTP supplémentaires. Il s'agit du nombre de comptes FTP pouvant être configurés pour accéder à l'abonnement, outre le compte FTP principal créé à l'activation de votre abonnement.

    Pour en savoir plus sur la configuration de comptes FTP, reportez-vous aux sections Changer les identifiants d'accès FTP et Ajouter des comptes FTP.

  • Bases de données (Linux). Il s'agit du nombre de bases de données pouvant être hébergées pour tous vos sites Web. Les bases de données sont des moyens standard d'organisation du stockage des données permettant aux sites Web dynamiques, applications Web et à leurs utilisateurs de stocker, rechercher et récupérer des informations. Pour en savoir plus, consultez le chapitre (Avancé) Utiliser les bases de données.
  • Bases de données MySQL et Bases de données du serveur Microsoft SQL (Windows). Il s'agit du nombre maximal de bases de données MySQL et Microsoft SQL Server pouvant être créées sur les serveurs de bases de données de Plesk et utilisées par les sites Web de l'abonnement.
  • Quota des bases de données MySQL et quota des bases de données Microsoft SQL (Windows). Il s'agit du total d'espace disque maximal en gigaoctets que les bases de données MySQL et Microsoft SQL Server de l'abonnement peuvent occuper.
  • Taille du fichier de bases de données MS SQL (hébergement Windows). Taille maximale du fichier d'une base de données Microsoft SQL Server. Remarque : si une base de données comporte plusieurs fichiers, elle peut occuper plus d'espace disque que vous ne l'avez indiqué ici.
  • Taille du fichier de log de la base de données MS SQL (hébergement Windows). Taille maximale du fichier de log d'une base de données Microsoft SQL Server. Remarque : si une base de données comporte plusieurs fichiers de log, elle peut occuper plus d'espace disque que vous ne l'avez indiqué ici.
  • Applications Java. Il s'agit du nombre d'applications Java packagées au format WAR que vous pouvez installer sur vos sites.
  • Sites publiés avec Presence Builder. Il s'agit du nombre de sites que vous pouvez créer et publier à l'aide de Presence Builder, si l'option Presence Builder est incluse dans votre pack d'hébergement.
  • Utilisateurs Web. Il s'agit du nombre de comptes utilisateur que vous pouvez créer pour les pages Web d'hébergement pour les autres utilisateurs sous vos domaines.
  • Connexions DSN ODBC. Il s'agit du nombre de connexions à des bases de données externes que vous pouvez créer pour les applications Web tournant sur votre compte client. Cette option n'est disponible que pour les comptes client Windows.
