Consulter les statistiques

Pour voir les rapports sur l'espace disque et l'utilisation du trafic par votre compte :

  1. Si plusieurs abonnements sont associés à votre compte, dans le menu Abonnement en haut de l'écran, sélectionnez l'abonnement requis.
  2. Allez dans Statistiques.

    Les informations suivantes sont présentées sous forme de graphiques :

    • Espace disque utilisé par les fichiers et répertoires suivants dans l'abonnement :
      • Sites Web
      • Comptes mail
      • Bases de données
      • Logs
      • Sauvegardes
      • Répertoires chroot
      • Fichiers de configuration
      • Répertoire FTP anonyme
    • Trafic des services FTP, Web, et de messagerie pendant le mois en cours.

      Le champ FTP présente des informations sur le volume total des fichiers transférés vers et à partir de l'abonnement via le protocole de transfert de fichiers.

      Le champ HTTP indique les informations sur le total de données transférées à partir de tous vos sites Web via le protocole HTTP, c'est-à-dire, récupérées par les navigateurs Web.

      Le champ POP3/IMAP indique le total de données reçues par tous les comptes mail sous vos domaines.

      Le champ SMTP indique le total de données envoyées par tous les comptes mail sous vos domaines.

  3. Exécutez l'une des actions suivantes :
    • Pour voir un rapport détaillé sur le total de données transférées vers ou à partir de vos sites via FTP, cliquez sur Statistiques FTP.
    • Pour voir un rapport détaillé sur le total de données transférées vers ou à partir de votre répertoire FTP, accessible sans autorisation, cliquez sur Statistiques FTP anonymes.
    • Pour voir un rapport sur le volume total du trafic des services sur un mois précis, cliquez sur Statistiques de transferts de données, et sélectionnez le mois voulu dans le menu.

Attention ! Selon la politique de votre fournisseur, votre abonnement risque d'être suspendu automatiquement en cas de surutilisation de l'espace disque. Appliquez ces quelques astuces pour libérer un peu d'espace disque :

  • Supprimez tous les fichiers inutiles et/ou obsolètes stockés dans le dossier httpdocs.
  • Configurez la rotation des logs comme expliqué dans la section Fichiers de log.
  • Supprimez les mails obsolètes ou configurez votre client de messagerie afin de télécharger les mails depuis le serveur en passant au POP3. Pour en savoir plus, consultez la section Accéder à votre boîte mail.
  • Supprimez les fichiers de sauvegarde obsolètes. Pour cela, consultez la section Envoyer et télécharger les fichiers de sauvegarde.
  • Supprimez les bases de données inutiles. Pour en savoir plus, consultez la section Gérer les bases de données du site Web.

Dans le graphique des services, vous pouvez voir l'utilisation de l'espace disque en fonction des contenus Web, mails, sauvegardes, logs et bases de données. Nous vous recommandons de vérifier les services qui consomment le plus d'espace disque. Si aucune de ces étapes ne vous a aidé, contactez votre fournisseur afin qu'il augmente la taille de l'espace disque disponible dans votre abonnement.
