État d'un site Web

L'état d'un site Web définit si le site est disponible via Internet. Il définit également quels sont les services d'hébergement fournis pour ce site. L'hébergeur peut avoir besoin de modifier l'état d'un site si le client ne paie pas les services. Les propriétaires de sites peuvent modifier l'état de leurs sites s'ils veulent les rendre inaccessibles temporairement.

Plesk propose trois états de sites Web : Suspendu, Désactivé et Actif.

Sites suspendus

Si vous voulez suspendre l'activité d'un site Web pour procéder à une maintenance et informer vos visiteurs que le site est temporairement inaccessible, vous pouvez utiliser l'état Suspendu. De cette manière, le site ne s'affiche pas dans les navigateurs (pour cela, allez dans Sites Web & Domaines > nom de domaine et cliquez sur Suspendre). Les visiteurs seront redirigés vers la page d'erreur "503 Service Unavailable" avec le code d'erreur HTTP 503. Le référencement du site par les moteurs de recherche n'est pas affecté et les services d'hébergement comme la messagerie sont toujours accessibles et gérables via Plesk.

Vous pouvez personnaliser la page d'erreur. Pour cela, cliquez sur le lien Modifier la page d'erreur disponible sous Panneau Client > Sites Web & Domaines > nom de domaine.

Remarque : le lien Modifier la page d'erreur s'affiche uniquement si le pack d'hébergement inclut l'option de personnalisation des documents d'erreur du serveur Web. Autrement dit, l'option Documents d'erreurs personnalisés est activée dans les paramètres du domaine, sous Sites Web & Domaines > nom de domaine > Paramètres d'hébergement.

Sites désactivés

Si vous arrêtez la maintenance d'un site Web, vous pouvez le désactiver en cliquant sur le lien Désactiver sous Sites Web & Domaines > nom de domaine. Les visiteurs verront la page par défaut du serveur Web définie par l'hébergeur et le site ne sera plus référencé par les moteurs de recherche.

Les sites Web désactivés ne sont plus hébergés sur le serveur : ils sont exclus de la configuration du serveur Web. Toutefois, les clients FTP et le Gestionnaire de fichiers peuvent accéder aux répertoires physiques et aux fichiers des sites désactivés. Les services d'hébergement tels que la messagerie ne seront pas disponibles.

Remarque : dans les éditions de Plesk antérieures à la version 11.5 cet état s'intitulait Suspendu.

Sites actifs

Pour remettre un site Web en ligne, allez sous Sites Web & Domaines > nom de domaine et cliquez sur Activer. Le site Web refonctionne alors normalement.
