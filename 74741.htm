Désactiver le service DNS de Plesk

Par défaut, les domaines créés dans Plesk utilisent le serveur DNS de Plesk pour la résolution des noms. Cependant, dans certaines situations, cela n'est pas souhaitable. Par exemple, si vous utilisez un serveur DNS tiers ou des serveurs de référence pour votre domaine, nous vous recommandons de désactiver le service DNS pour le domaine. Pour cela, allez sous Sites Web & Domaines > Paramètres DNS et cliquez sur Désactiver. Si vous souhaitez le réactiver ultérieurement, cliquez sur Activer ; le service DNS de Plesk sera réactivé pour domaine.

Quand le service DNS de Plesk d'un domaine est activé, vous pouvez vérifier si les enregistrements DNS stockés sur les serveurs DNS tiers sont corrects. Pour cela, allez sous Sites Web & Domaines > Paramètres DNS, cliquez sur Ajouter un enregistrement. Saisissez le nom de domaine du serveur DNS tiers et cliquez sur l'icône en regard du nom du serveur DNS que vous venez d'ajouter. Plesk récupère alors la liste des enregistrements DNS depuis le serveur DNS tiers, compare cette liste aux enregistrements DNS du domaine, stockés dans la base de données Plesk et vous avertit en cas de différences.
