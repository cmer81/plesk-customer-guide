Gérer les versions de Ruby via l'interface de ligne de commande

Si vous avez accès à votre serveur Plesk via SSH, vous pouvez gérer les versions de Ruby par la ligne de commande. Les commandes doivent être exécutées depuis le répertoire d'application principal, c'est-à-dire /var/www/vhosts/<nom_domaine>/<nom_application>, où <nom_domaine> est le nom de domaine du site Web et <nom_application> est le nom de l'application Ruby.

Plesk essaie de rester synchronisé avec la version sélectionnée de Ruby pour l'application. Exécutez la commande suivante pour vérifier la version active de Ruby :

ruby –v

Vous pouvez aussi utiliser la commande rbenv pour la gestion des versions de Ruby. Pour plus d'informations, reportez-vous au lien https://github.com/rbenv/rbenv#command-reference).
