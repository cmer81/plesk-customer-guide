Créer un alias de messagerie

Voir le tutoriel vidéo

Pour ajouter ou supprimer les adresses mail (ou alias de messagerie) pour un compte mail :

  1. Allez dans Mail > adresse mail > Alias de messagerie.
  2. Exécutez l'une des actions suivantes :
    • Pour ajouter une adresse, saisissez-la dans le champ Alias de messagerie et cliquez sur OK.
    • Pour supprimer une adresse, cliquez sur le lien Supprimer à droite de l'adresse que vous voulez supprimer.
