Démarrer rapidement avec Plesk

Ce chapitre s'adresse aux débutants en matière d'hébergement Web. Si vous créez un site Web pour la première fois, commencez par lire ce chapitre.

Voici les avantages d'une présence Web pour votre entreprise :

  • Donner une meilleure visibilité à votre entreprise : autrement dit, les clients vous trouveront plus facilement.
  • Faire de la publicité pour votre entreprise et communiquer des informations importantes.
  • Recevoir du feedback.

Vous disposez d'un certain nombre de possibilités pour présenter votre société sur le Web. La présence Web peut être aussi simple qu'une page de contacts ou aussi complexe que le site Web d'une grande entreprise avec l'accès à un système ERP. Mettre en ligne votre entreprise, c'est un véritable jeu d'enfant. Il vous suffit de suivre ces trois étapes :

  • Enregistrer un nom de domaine.
  • Acheter l'hébergement Web auprès d'un hébergeur.
  • Créer un site Web.

Ces trois étapes sont détaillées ci-dessous.

Enregistrer un nom de domaine

Votre nom de domaine s'assimile à l'adresse de votre entreprise. Vos clients l'utiliseront pour vous trouver en ligne, choisissez-le bien. Les meilleurs noms de domaine sont courts, simples à saisir et faciles à retenir. Par exemple : www.exemple.com. Vous pouvez enregistrer votre nom de domaine auprès de l'un des nombreux bureaux d'enregistrements, appelés "registrars de domaines". En général, un hébergeur peut également vous aider à enregistrer un nom de domaine. Les services d'hébergement Web sont souvent proposés sous forme de bundle avec des offres d'enregistrement de noms de domaine, ou l'inverse.

Attention : lorsque vous enregistrez votre nom de domaine via votre hébergeur, vérifiez si le nom de domaine est enregistré à votre nom. Dans le cas contraire, si vous veniez à changer d'hébergeur, vous risqueriez d'avoir des problèmes.

Acheter un compte client chez un hébergeur

Acheter un compte client, c'est un peu comme louer les locaux de votre entreprise : vous louez l'espace et l'infrastructure qui hébergent votre site Web. Cela inclut la connexion Internet, l'espace disque pour héberger le contenu de votre site Web, etc. Pour en savoir plus sur les comptes client dans Plesk, consultez le chapitre Administrer un compte client.

Créer votre site Web

Il existe trois méthodes pour créer le contenu de votre site Web. Chacune a ses inconvénients et ses avantages. Voici un résumé :

  • Presence Builder. L'éditeur Presence Builder fourni avec Plesk vous permet de créer des sites Web à l'aide d'une interface Web. Vous pouvez utiliser l'un des templates fournis pour créer un site Web professionnel en quelques clics seulement.
  • Systèmes de gestion de contenu tiers (CMS). Les Systèmes de gestion de contenu tiers (CMS) sont des applications tierces qui vous permettent de créer et de maintenir un site Web. Les CMS sont polyvalents et comportent de nombreux add-ons en option. Ils sont plus flexibles certes plus flexibles que Presence Builder, en termes de personnalisation. En revanche, ils demandent plus de connaissances techniques.
  • Designer Web et envoi du contenu. C'est la garantie d'obtenir exactement ce que vous voulez. Toutefois, cette solution est également la plus coûteuse. Le designer Web vous fournira les fichiers que vous devrez envoyer sur votre compte d'hébergement. Vous pouvez utiliser le FTP ou le Gestionnaire de fichiers.

Dans ce chapitre, nous passerons en revue ces trois méthodes de création de sites Web. Vous découvrirez également comment créer des boîtes mail pour les utilisateurs du site et comment afficher les statistiques de visites.

Fonctions d'hébergement avancées

Lorsque vous vous serez familiarisé avec les fonctions de base de Plesk, essayez d'utiliser les fonctions avancées : par exemple, installez des applications Web, sécurisez vos sites avec des certificats SSL/TLS, utilisez des bases de données, etc.

Remarque : les hébergeurs peuvent désactiver certaines fonctions avancées pour simplifier votre Panneau Client et le rendre plus convivial. Nous avons utilisé la mention (Avancé) pour indiquer les sections traitant des fonctionnalités pouvant être désactivées. Si vous avez besoin de l'une des options avancées, contactez votre hébergeur.
