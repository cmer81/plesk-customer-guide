Protéger votre messagerie Web avec un certificat SSL/TLS

Regardez le tutoriel vidéo

Vous pouvez utiliser une connexion sécurisée pour accéder à votre boîte mail via la messagerie Web.

Allez dans Sites Web & Domaines et cliquez sur Certificats SSL/TLS. Sélectionnez un certificat dans la liste et cliquez sur Protéger la messagerie Web. Le certificat est alors marqué comme "Utilisé pour protéger la messagerie Web".

Par la suite, la connexion HTTPS sera utilisée lorsque vous allez sous Mail et cliquez sur l'icône de la messagerie Web. Pour plus d'informations, voyez la section Accéder à la messagerie Web).

Remarque : cette fonctionnalité n'est pas disponible sur les OS ne prenant pas en charge SNI. Vous trouvez plus d'informations dans l'article SSL et adresses IP partagées).
