Amazon aStore

Vous souhaiterez peut-être ajouter une boutique en ligne Amazon aStore afin de vendre des produits disponibles chez Amazon via votre site. Avec l'application Amazon aStore, vous pouvez créer facilement une boutique en ligne professionnelle qui s'intègre à votre site Web. Pour en savoir plus sur la boutique en ligne aStore, consultez la page Web http://astore.amazon.com/.

Pour ajouter un bloc Amazon aStore :

  1. Rejoignez le Club Partenaires Amazon depuis la page https://affiliate-program.amazon.com/. Créez un aStore et copiez le lien fourni.
  2. Allez dans l'onglet Modules, sélectionnez Amazon aStore et faites glisser le module vers la page.
  3. Dans le champ, collez le code que vous avez obtenu d'Amazon aStore.
  4. Cliquez sur OK.

L'Amazon aStore s'affichera sur votre site Web.

Pour supprimer un bloc Amazon aStore :

Placez le pointeur de la souris sur le bloc et cliquez sur Supprimer.
