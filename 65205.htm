Télécharger les fichiers de sauvegarde de la base de données (Windows)

À compter de Plesk Onyx, le stockage de sauvegarde de la base de données est disponible uniquement si vous avez créé des sauvegardes de la base de données dans les versions antérieures de Plesk. Vous pouvez uniquement supprimer ou télécharger vos sauvegardes. Pour sauvegarder des bases de données, utilisez les options d'export et d'import dans Sites Web & Domaines > Bases de données.

Pour télécharger un fichier de sauvegarde depuis le stockage de sauvegarde :

  1. Allez dans Sites Web & Domaines > Gestionnaire de sauvegardes > Plus d'actions > Stockage de sauvegarde de bases de données.
  2. Sélectionnez la base de données qui contient les fichiers de sauvegarde que vous voulez parcourir dans le menu Base de données. Ne décochez pas l'option Toutes les bases de données du domaine si vous voulez parcourir les fichiers de sauvegarde de toutes les bases de données sur un domaine.
  3. Cliquez sur l'icône correspondant au fichier de sauvegarde de la base de données que vous voulez télécharger.
  4. Indiquez où vous voulez enregistrer le fichier de sauvegarde, puis cliquez sur Sauvegarder.

Le fichier de sauvegarde sera téléchargé depuis le stockage de sauvegarde.

Pour supprimer un fichier de sauvegarde depuis le stockage de sauvegarde :

  1. Allez dans Sites Web & Domaines > Gestionnaire de sauvegardes > Plus d'actions > Stockage de sauvegarde de bases de données.
  2. Sélectionnez la base de données qui contient les fichiers de sauvegarde que vous voulez supprimer dans le menu Base de données. Ne décochez pas l'option Toutes les bases de données du domaine si vous voulez parcourir les fichiers de sauvegarde de toutes les bases de données sur un domaine.
  3. Cochez la case correspondant au fichier de sauvegarde de la base de données que vous voulez supprimer. Cliquez sur Supprimer, confirmez la suppression, puis cliquez sur OK.
