Modifier votre mot de passe et vos coordonnées

Regardez le tutoriel vidéo (Modifier le mot de passe de votre abonnement)
Regardez le tutoriel vidéo (Mettre à jour les informations de contact)

Pour modifier le mot de passe qui vous permet d'accéder au Panneau Client :

  1. Cliquez sur un lien incluant votre nom en haut de l'écran.
  2. Cliquez sur Éditer le profil.
  3. Dans les paramètres du profil, saisissez un nouveau mot de passe et confirmez-le, puis cliquez sur OK.

Pour changer le nom d'utilisateur ou mot de passe que vous utilisez pour vous connecter à votre abonnement via FTP ou SSH :

  1. Allez dans Sites Web & Domaines.
  2. Cliquez sur Paramètres d'hébergement Web.
  3. Saisissez le nouveau nom d'utilisateur ou mot de passe, puis cliquez sur OK.

Pour changer vos coordonnées :

  1. En haut de l'écran, cliquez sur le lien avec votre nom.
  2. Cliquez sur Éditer le profil.
  3. Cliquez sur l'onglet Détails du contact.
  4. Mettez à jour vos informations de contact et cliquez sur OK.
