Plesk en tant que Serveur DNS esclave

Si vous hébergez des sites Web sur votre compte et avez un serveur DNS autonome qui fait office de serveur de noms primaire (maître) pour vos sites, vous voudrez probablement configurer le serveur DNS de Plesk pour qu'il serve de serveur de noms secondaire (esclave).

Pour désigner le serveur DNS de Plesk comme serveur de noms secondaire :

  1. Allez dans Sites Web & Domaines et cliquez sur le nom de domaine dont vous voulez gérer les paramètres DNS.
  2. Cliquez sur Paramètres DNS.
  3. Cliquez sur Maître/Esclave pour changer le mode du serveur DNS.
  4. Cliquez sur Ajouter un enregistrement.
  5. Indiquez l'adresse IP du serveur DNS primaire (maître).
  6. Cliquez sur OK, puis sur Mettre à jour.
  7. Répétez les étapes 2 à 6 pour chaque site Web nécessitant un serveur de noms secondaire sur votre serveur.

Pour redéfinir le serveur DNS de Plesk comme serveur primaire :

  1. Allez dans Sites Web & Domaines et cliquez sur le nom de domaine dont vous voulez gérer les paramètres DNS.
  2. Cliquez sur Paramètres DNS.
  3. Cliquez sur Maître/Esclave pour changer le mode du serveur DNS.

    Les enregistrements de ressources d'origine de la zone seront restaurés.
