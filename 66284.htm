Éditer des sites Web

Dans Presence Builder, vous pouvez changer quasiment chaque élément de votre site Web. La plupart des éléments sont changés sur place : vous pouvez ouvrir la page où l'élément est utilisé, l'éditer et voir le résultat directement. Les modifications apportées aux éléments valables pour l'ensemble du site, tels que ceux insérés dans les zones DESIGN, sont centralisées, mais appliquées à toutes les pages de votre site.

Remarque : si vous utilisez Internet Explorer, nous vous recommandons vivement de ne pas utiliser le zoom du navigateur pour changer la taille du texte qui s'affiche dans la fenêtre de votre navigateur. Notez que l'affichage des sites Web dans Presence Builder sera meilleur en zoom normal (100 %).
