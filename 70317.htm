Créer des sites Web avec Presence Builder

Presence Builder (également appelé l'éditeur) est un éditeur visuel qui permet aux propriétaires des petites entreprises et particuliers de créer leurs sites Web, et ce rapidement et facilement. Avec Presence Builder, vous n'avez pas besoin d'écrire de code ou de configurer des serveurs pour commencer un blog, ouvrir une boutique en ligne ou configurer toute autre forme de site Web. Il vous suffit d'ajouter des blocs de contenus qui incluent votre site (texte, galerie d'images, boutique en ligne) en les faisant glisser sur les pages, de compléter ces blocs par votre contenu et de publier le résultat. Et voilà, c'est tout !

Création d'un site Web de A à Z

Lorsque vous commencez à travailler sur votre site Web, vous n'avez pas besoin d'écrire du texte, de rechercher des images et d'adapter l'apparence du site. Il vous suffit de choisir le thème le plus adapté à votre site Web et Presence Builder générera un template de site qui contient déjà un jeu de pages, d'images et de textes, le tout selon le thème sélectionné. Chaque fois que Presence Builder génère le design d'un site Web, il utilise tout un jeu aléatoire de designs et d'autres composants. Résultat : vous pouvez être sûr que le design de votre site Web est unique.

Une fois que vous avez sélectionné ce thème, vous êtes invité à entrer d'autres informations que Presence Builder ajoute aux pages du site Web, par exemple, votre numéro de téléphone qui apparaîtra sur la page Nous contacter.

Pour en savoir plus sur la procédure pour créer votre site Web, consultez la section Créer un site Web.

Importer des sites Web créés dans SiteBuilder 4.5.

Si vous avez déjà créé un site Web avec SiteBuilder 4.5 et l'avez hébergé autre part, vous pouvez l'importer vers votre nouveau compte client. Pour en savoir plus sur la migration de votre site Web SiteBuilder 4.5 vers Presence Builder, consultez la section Importer les sites de SiteBuilder 4.5.

Enrichir le site avec du contenu.

Dès que Presence Builder crée votre site Web, vous pouvez le compléter avec votre contenu. Votre outil d'édition majeur est le menu principal de Presence Builder. Depuis ce menu, vous pouvez :

  • Modifier la structure de votre site : ajouter ou supprimer des pages.
  • Modifier le design du site Web : modifier les couleurs et la mise en page.
  • Mettre votre contenu sur le site : supprimer du contenu inutile et ajouter vos propres éléments :
    • Texte
    • Images et galeries d'images
    • Vidéos
    • Scripts
    • Blogs
    • Champs de recherche
    • Autres éléments
  • Permettre à vos visiteurs de partager les informations de votre site Web sur un réseau social tel que Facebook ou Twitter : ajouter les boutons correspondants sur les pages de votre site.

Pour en savoir plus sur les actions disponibles dans l'éditeur, consultez la section Se familiariser avec Presence Builder.

Trouvez les instructions sur la gestion de l'apparence et du contenu dans la section Éditer un site Web.

Rendre le site accessible sur Internet

Après avoir terminé l'édition du site Web, publiez-le sur Internet. Pour permettre aux visiteurs de voir votre site via votre nom de domaine, publiez le site Web en cliquant sur le bouton correspondant dans le menu principal. Lorsque vous cliquez sur ce lien, Presence Builder génère les pages statiques et les scripts qui incluent votre site Web. Il les place ensuite aux emplacements appropriés sur le serveur afin que les visiteurs puissent les voir lorsqu'ils tapent votre nom de domaine dans leur navigateur. Pour en savoir plus sur la publication des sites Web, consultez la section Publier un site Web via Internet.

Vous pouvez également attirer davantage de visiteurs sur vos sites en publiant une copie sur Facebook, l'un des réseaux sociaux les plus populaires. Consultez la section Publier une copie du site Web dans Facebook pour en savoir plus sur cette fonctionnalité.
