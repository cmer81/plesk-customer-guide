Protéger votre site Web avec un certificat SSL/TLS

Pour protéger votre site Web avec un certificat SSL/TLS, vous pouvez utiliser un certificat déjà en votre possession.

Vous avez d'abord besoin de charger le certificat. Pour ce faire, allez sous Sites Web & Domaines et cliquez sur Certificats SSL/TLS > Ajouter un certificat SSL/TLS.

Saisissez un nom de certificat (vous l'utiliserez pour reconnaître le certificat), puis envoyez-le comme suit.

  • Si votre certificat est stocké sous forme de fichier *.key et *.crt faites défiler l'écran jusqu'à la section Envoyer les fichiers du certificat et envoyez les fichiers. Si le certificat et les clés privées de votre certificat sont dans un fichier *.pem, envoyez votre fichier deux fois : une fois en tant que clé privée et une fois en tant que certificat. Cliquez sur Charger le certificat quand vous avez terminé.
  • Si vous stockez le certificat au format texte, faites défiler l'écran jusqu'à la section Envoyer le certificat en mode texte et collez le certificat et les clés privées dans les champs correspondants. Cliquez sur Charger le certificat quand vous avez terminé.

Le certificat sera alors placé dans votre répertoire. Pour voir la liste de tous les certificats SSL/TLS dans votre répertoire, allez sous Sites Web & Domaines > Certificats SSL/TLS.

Maintenant que le certificat est généré, vous devez l'installer. Allez sous Sites Web & Domaines et cliquez sur Paramètres d'hébergement. Cochez la case Prise en charge SSL/TLS , sélectionnez le certificat que vous venez de charger depuis le menu Certificat et cliquez sur OK.
