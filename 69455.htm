Paramètres d'hébergement

Les paramètres d'hébergement par site Web sont regroupés comme suit :

  • Général.
    Paramètres de type d'hébergement, sécurité, script et statistiques. Voir Paramètres généraux.
  • Script Web (PHP ou ASP.NET).
    Paramètres du langage de script PHP. Consultez les sections Configuration de PHP ou Paramètres ASP.NET.
  • Serveur Web (Apache & Nginx ou IIS)
    Paramètres du serveur Web. Le type de serveur Web dépend du système d'exploitation de votre serveur Plesk : Apache (avec Nginx) est utilisé sur Plesk pour Linux et IIS sur Plesk pour Windows. Voir respectivement les sections Paramètres du serveur Web Apache et Paramètres du serveur Web IIS.

    Remarque : dans les paramètres du serveur Web IIS vous pouvez configurer les paramètres de sécurité de base du site Web, comme l'accès anonyme, l'utilisation de SSL/TLS et l'exploration des répertoires.

Sur Windows, vous pouvez également définir des paramètres ASP.NET personnalisés dans Sites Web & Domaines > sélectionner un domaine > Paramètres ASP.NET. Pour en savoir plus, consultez la section Paramètres ASP.NET (Windows).
