Autres services

Certaines applications Web associées à la messagerie (comme la messagerie Web ou le filtre anti-spam) peuvent partager certaines de leurs fonctions directement dans votre Panneau Client. Par exemple, vous pouvez associer un compte mail à un compte dans la messagerie Web Open-Xchange. Dans ce cas, l'utilisateur de la messagerie verra le lien vers la messagerie Web dans son Panneau Client.

Afin d'intégrer une application de cette manière, vous devez l'autoriser à accéder à un compte mail spécifique. Vous pouvez configurer cela dans les paramètres du compte mail pendant la création ou l'édition du compte.

Pour autoriser certaines applications à accéder à votre compte mail :

  1. Allez dans Mail > sélectionner une adresse mail > onglet Autres services.
  2. Dans la liste des applications installées qui fournissent des services de messagerie, sélectionnez l'option Accordé pour les applications qui doivent accéder à ce compte mail.
  3. Cliquez sur OK.

Pour refuser l'accès à votre compte mail à certaines applications :

  1. Allez dans Mail > sélectionner une adresse mail > onglet Autres services.
  2. Dans la liste des applications installées qui fournissent des services de messagerie, sélectionnez l'option Refusé pour les applications qui ne doivent pas accéder à ce compte mail.
  3. Cliquez sur OK.
