Envoyer, télécharger et supprimer des fichiers de sauvegarde

Vous pouvez télécharger sur votre ordinateur les fichiers de sauvegarde que vous créez dans Plesk. La redondance qui en résulte peut vous donner une certaine tranquillité d’esprit, ou vous pouvez réduire le volume d’espace disque que vous utilisez en supprimant dans Plesk les sauvegardes que vous avez sauvegardées localement. Vous pouvez envoyer dans Plesk les fichiers de sauvegarde de votre ordinateur à n’importe quel moment.

Télécharger des fichiers de sauvegarde à partir de Plesk

Pour télécharger un fichier de sauvegarde de Plesk, allez sous Sites Web & Domaines > Gestionnaire de sauvegardes, cliquez sur la sauvegarde que vous souhaitez télécharger, puis cliquez sur Télécharger.

Avant le téléchargement du fichier, Plesk vous invite à protéger la sauvegarde avec un mot de passe. Cela chiffre les mots de passe enregistrés dans la base de données Plesk. De cette façon, même si le fichier de sauvegarde tombe dans les mains d’un tiers, il sera impossible d’obtenir les mots de passe enregistrés à l’intérieur, car restaurer un fichier protégé par un mot de passe sans fournir le bon mot de passe a pour résultat que tous les mots de passe sont générés de façon aléatoire. Si vous ne souhaitez pas protéger votre sauvegarde par un mot de passe, décochez la case Utiliser la protection par mot de passe.

Pour terminer l’opération, cliquez sur OK. Le fichier de sauvegarde sera téléchargé dans le répertoire "Téléchargements" spécifié dans vos paramètres de navigateur.

Charger les fichiers de sauvegarde dans Plesk

Pour envoyer un fichier de sauvegarde dans Plesk, allez sous Sites Web & Domaines > Gestionnaire de sauvegardes puis cliquez sur Envoyer. Cliquez sur Choisir le fichier et naviguez jusqu’à l’endroit où le fichier de sauvegarde que vous souhaitez envoyer est enregistré.

Si la sauvegarde a été protégée par un mot de passe quand elle a été téléchargée, vous devez taper ce dernier dans le champ correspondant lorsque vous envoyez la sauvegarde. Si vous ne le faites pas, ou si le mot de passe que vous entrez est incorrect, la sauvegarde va certes êtes envoyée, mais toute tentative de la restaurer sans fournir le bon mot de passe réinitialisera tous les mots de passe contenus dans la sauvegarde, comme les mots de passe des boîtes mail, les mots de passe de l’utilisateur de la base de données, etc.

Pour terminer l’opération, cliquez sur OK. Le fichier de sauvegarde sera envoyé dans le répertoire Plesk.

Supprimer les fichiers de sauvegarde

Pour supprimer un ou plusieurs fichiers de sauvegarde, allez sous Sites Web & Domaines > Gestionnaire de sauvegardes, cochez les cases correspondant aux fichiers de sauvegarde que vous souhaitez supprimer et cliquez sur Supprimer.

La suppression de toute une sauvegarde supprime automatiquement toutes les sauvegardes incrémentables qui l’utilisent comme sauvegarde parent.

Attention : toutes les informations enregistrées dans les fichiers de sauvegarde supprimés sont perdues de façon permanente. Il n’y a pas de bouton "Annuler". Assurez-vous de bien supprimer uniquement les sauvegardes dont vous n’avez plus besoin.
