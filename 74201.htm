Accéder aux bases de données avec les chaînes de connexion

Pour vous connecter à une base de données, vous pouvez aussi utiliser une chaîne de connexion. Une chaîne de connexion est une chaîne qui contient des informations sur la source des données (en général le moteur d'une base de données) ainsi que les informations requises pour s'y connecter. Si la syntaxe exacte d'une source de connexion varie selon la source des données (par ex. : une chaîne de connexion qui doit se connecter à un serveur de base de données MySQL sera différente d'une chaîne de connexion pour un serveur PostgreSQL), vous devrez toujours préciser les données suivantes :

  • Adresse du serveur de base de données. Pour trouver l'adresse de votre serveur de base de données, procédez comme suit :
    1. Allez dans Sites Web & Domaines et cliquez sur Bases de données.
    2. Cliquez sur le lien Admin Web pour l'une de vos bases de données.
    3. En haut de l'écran, cherchez une entrée du type : "Serveur : 10.58.103.102:3306". Dans cet exemple, "10.58.103.102" correspond à l'adresse du serveur de base de données. Si le nom du serveur indiqué est "localhost", contactez votre hébergeur pour trouver l'adresse de votre serveur de base de données.
  • Nom de la base de données. Pour déterminer le nom exact de votre base de données, allez dans Sites Web & Domaines, puis dans Bases de données pour afficher la liste de vos bases de données.
  • Nom d'utilisateur et mot de passe de l'utilisateur de la base de données. Pour déterminer le nom d'utilisateur de l'utilisateur de la base de données, procédez comme suit :
    1. Allez dans Sites Web & Domaines et cliquez sur Bases de données.
    2. Notez le nom indiqué dans la colonne Utilisateurs, à côté de la base de données à laquelle vous voulez vous connecter à l'aide de la chaîne de connexion.

    Si vous ne vous rappelez pas le mot de passe de l'utilisateur, vous pouvez le réinitialiser. Pour réinitialiser le mot de passe de l'utilisateur, procédez comme suit :

    1. Allez dans Sites Web & Domaines et cliquez sur Bases de données.
    2. Cliquez sur Gestion des utilisateurs, puis sur le nom correspondant dans la colonne Nom.
    3. Saisissez le nouveau mot de passe et confirmez-le. Cliquez sur OK.

L'objectif de ce document n'est pas de fournir une description approfondie de la syntaxe "chaîne de connexion". Pour en savoir plus, nous vous recommandons de consulter la page https://msdn.microsoft.com/en-us/library/windows/desktop/ms722656%28v=vs.85%29.aspx.
