Protéger les connexions avec des certificats SSL/TLS

Si votre site Web accepte les données confidentielles (comme les informations de cartes bancaires), nous vous recommandons de sécuriser les communications vers et depuis votre site à l'aide d'un certificat SSL/TLS. Un certificat SSL/TLS présente des avantages à deux niveaux :

  • Toutes les connexions depuis et vers votre site Web sont chiffrées. Elles ne peuvent pas être interceptées par des tiers.
  • Un certificat SSL/TLS émis pour votre site Web confirme la validité de votre site. Ainsi, les visiteurs savent qu'ils consultent bien votre site Web (et non un site Web frauduleux créé par un tiers).

Si les visiteurs de votre site savent que votre site est authentique et que leurs informations de paiement ne sont pas compromises, ils ont confiance en votre site. Ainsi, un certificat SSL/TLS est essentiel, pour une boutique en ligne par exemple.

Si vous voulez protéger votre site Web avec un certificat SSL/TLS, vous pouvez le faire via Plesk. Choisissez l'une de ces possibilités :

  • Utiliser un certificat que vous possédez déjà.
  • Obtenir un certificat gratuit de Let's Encrypt.
  • Acheter un certificat via Plesk.
  • Acheter un certificat auprès d'une autorité de certification.
  • Utiliser un certificat auto-signé.

Pour gérer vos certificats SSL/TLS, allez dans Sites Web & Domaines et cliquez sur Certificats SSL/TLS.
